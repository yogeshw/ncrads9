Analysis Tools

NCRADS9 provides a suite of analysis tools for examining your astronomical data.

Statistics

Access via Analysis → Statistics

The statistics tool computes for a selected region or the entire image:

Statistic Description
Sum Total of all pixel values
Mean Average pixel value
Median Median pixel value
Std Dev Standard deviation
Min/Max Minimum and maximum values
Centroid Intensity-weighted center

Histogram

Access via Analysis → Histogram

The histogram tool displays the distribution of pixel values. Features include:

  • Linear or logarithmic Y-axis
  • Adjustable number of bins
  • Region-based histograms
  • Export data to file

Radial Profile

Access via Analysis → Radial Profile

Create radial profiles around a center point to analyze:

  • Point spread function (PSF)
  • Galaxy surface brightness profiles
  • Extended source structure

Coordinate Systems

NCRADS9 supports multiple coordinate systems:

System Description
Image Pixel coordinates (1-indexed)
Physical Physical pixel coordinates
FK5 J2000 equatorial (RA/Dec)
FK4 B1950 equatorial
Galactic Galactic coordinates (l/b)
Ecliptic Ecliptic coordinates

Switch coordinate systems via WCS menu or the coordinate dropdown.

Contours

Access via Analysis → Contours

Overlay contour lines on your image:

  • Automatic or manual level selection
  • Linear, logarithmic, or custom spacing
  • Customizable colors and line styles
  • Save contour parameters

Cross-sections

Create 1D cross-sections through your image data by drawing a line region and selecting Analysis → Cross-section.
