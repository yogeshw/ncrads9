Analysis Tools

NCRADS9 provides a suite of analysis tools for examining your astronomical data.

Statistics

Access via Analysis → Statistics

The statistics dialog computes summary values for the loaded image:

Statistic Description
Sum Total of all pixel values
Mean Average pixel value
Median Median pixel value
Std Dev Standard deviation
Min/Max Minimum and maximum values

Histogram

Access via Analysis → Histogram

The histogram dialog displays the distribution of valid pixel values (NaN/Inf excluded) using a fixed binning and includes a quick summary of min/max/mean/median.

Radial Profile

A radial-profile menu entry exists, but UI integration is still in progress in the current release.

Coordinate Systems

NCRADS9 supports WCS coordinate display in these systems:

System Description
ICRS International Celestial Reference System
FK5 J2000 equatorial (RA/Dec)
FK4 B1950 equatorial
Galactic Galactic coordinates (l/b)
Ecliptic Ecliptic coordinates

Switch coordinate systems via the WCS menu.

Contours

Access via Analysis → Contours

Overlay contour lines on your image:

  • Automatic or manual level selection
  • Linear, logarithmic, or custom spacing
  • Customizable colors and line styles
  • Export contour levels and paths to JSON
