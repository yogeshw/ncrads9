Analysis Tools

NCRADS9 provides a DS9-style Analysis menu for examining and inspecting astronomical images.

Statistics

Access via Analysis → Statistics

The statistics dialog computes summary values for the loaded image:

Statistic Description
Sum Total of all pixel values
Mean Average pixel value
Median Median pixel value
Std Dev Standard deviation
Min/Max Minimum and maximum values

Histogram

Access via Analysis → Histogram

The histogram dialog displays the distribution of valid pixel values (NaN/Inf excluded) using a fixed binning and includes a quick summary of min/max/mean/median.

Radial Profile

Access via Analysis → Radial Profile

Plots the mean radial profile around the image center in an interactive pop-up plot.

Coordinate Systems

NCRADS9 supports WCS coordinate display in these systems:

System Description
ICRS International Celestial Reference System
FK5 J2000 equatorial (RA/Dec)
FK4 B1950 equatorial
Galactic Galactic coordinates (l/b)
Ecliptic Ecliptic coordinates

Switch coordinate systems via the WCS menu.

Contours

Access via Analysis → Contours and Analysis → Contour Parameters

Overlay contour lines on your image:

  • Automatic or manual level selection
  • Linear, logarithmic, or custom spacing
  • Customizable colors and line styles
  • Export contour levels and paths to JSON

Block and Smooth

The Analysis menu includes DS9-style Block controls (Block In/Out/Fit and block factors 1-32) and a Smooth toggle with smoothing parameters.

Coordinate Grid and Name Resolution

Analysis → Coordinate Grid and Coordinate Grid Parameters expose grid settings, and Analysis → Name Resolution resolves object names to sky coordinates and reports/pans when WCS is available.

Mask, Crosshair, and Graph Tools

Mask Parameters provides finite/range masking for analysis products, Crosshair Parameters controls an interactive crosshair overlay, and Graph Parameters controls horizontal/vertical profile graph panels.

Analysis Command Loading

Load Analysis Commands can import simple external command lists (`label|command` format). Loaded entries appear in the Analysis menu and can be cleared using Clear Analysis Commands.
