Colormaps and Scale

Available Colormaps

NCRADS9 now exposes DS9-style color controls in the top-level Color menu, including default astronomical maps, grouped scientific maps, and user-loaded colormaps.

Colormaps
Grey, Red, Green, Blue, A, B, BB, HE, I8, AIPS0, SLS, HSV, Heat, Cool, Rainbow, Standard, Staircase, Color, plus grouped Matplotlib Uniform maps (Viridis/Plasma/Inferno/Magma) and user-loaded maps

Access colormaps via the top-level Color menu, including User → Load Colormap... for .lut/.sao files.

Colorbar Controls

The Color menu includes colorbar controls for visibility, orientation, numerics, spacing mode, font size, bar size, and tick count.

Scale Options

The scale determines how pixel values map to display colors:

Scale Description Best For
Linear Direct mapping of values Uniform data ranges
Log Logarithmic mapping Large dynamic range
Sqrt Square root mapping Moderate dynamic range
Squared Squared mapping Enhancing bright features
Asinh Inverse hyperbolic sine Wide range with negatives
Histogram Equalization Equalizes histogram Maximum detail

Access scale options via Scale menu.

Contrast and Bias

Fine-tune the display using contrast and bias controls:

  • Contrast: Adjusts the range of values displayed (vertical mouse drag)
  • Bias: Shifts the center of the display range (horizontal mouse drag)

Keyboard Shortcuts

  • Right-click + drag: Adjust contrast and bias interactively

Scale Limits

Set specific minimum and maximum values via Scale → Scale Parameters:

  • Min/Max: Uses full data range
  • ZScale: IRAF zscale algorithm
  • User: Manual specification
  • Percentile: Based on data percentiles
