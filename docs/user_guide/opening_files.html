Opening FITS Files

Supported Formats

NCRADS9 supports the following file formats:

Format Extensions Description
FITS .fits, .fit, .fts Flexible Image Transport System
Compressed FITS .fits.gz, .fits.fz Gzip or fpack compressed FITS
FITS Array .fits FITS files with array extensions

Loading Images

There are several ways to open a FITS file:

From the Menu

Select File → Open or use the keyboard shortcut Ctrl+O.

From the Command Line

ncrads9 /path/to/image.fits

Drag and Drop

Simply drag a FITS file onto the NCRADS9 window.

Multi-Extension FITS

For FITS files with multiple extensions, NCRADS9 will display an extension selector dialog. You can:

  • Select a specific extension to view
  • Navigate between extensions using Frame → Next and Frame → Previous
  • Use the extension dropdown in the toolbar

Opening Multiple Files

You can open multiple FITS files simultaneously:

ncrads9 file1.fits file2.fits file3.fits

Each file will open in a separate frame, which you can tile or cascade using the Window menu.
