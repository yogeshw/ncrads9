Getting Started with NCRADS9

Installation

NCRADS9 can be installed using pip:

pip install ncrads9

Or install from source:

git clone https://github.com/ncra/ncrads9.git
cd ncrads9
pip install -e .

Requirements

  • Python 3.8 or later
  • NumPy
  • Astropy
  • PyQt5 or PyQt6

Launching NCRADS9

Once installed, you can launch NCRADS9 from the command line:

ncrads9

To open a FITS file directly:

ncrads9 image.fits

Basic Navigation

  • Zoom: Use the mouse scroll wheel to zoom in and out
  • Pan: Click and drag to pan across the image
  • Reset View: Press Home to reset the view
  • Full Screen: Press F11 to toggle full screen mode

Interface Overview

The NCRADS9 interface consists of:

  • Main Display: The central area showing the FITS image
  • Menu Bar: Access to file operations, view options, and analysis tools
  • Toolbar: Quick access buttons for common operations
  • Status Bar: Displays cursor position and pixel values
