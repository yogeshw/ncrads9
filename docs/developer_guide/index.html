NCRADS9 Developer Guide

This guide provides information for developers who want to contribute to NCRADS9 or extend its functionality.

Setting Up the Development Environment

Clone the Repository

git clone https://github.com/ncra/ncrads9.git
cd ncrads9

Create a Virtual Environment

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

Install Development Dependencies

pip install -e ".[dev]"

Project Structure

ncrads9/
├── ncrads9/           # Main package
│   ├── __init__.py
│   ├── core/          # Core functionality
│   ├── regions/       # Region handling
│   ├── analysis/      # Analysis tools
│   └── ui/            # User interface
├── tests/             # Test suite
├── docs/              # Documentation
├── resources/         # Static resources
└── pyproject.toml     # Project configuration

Running Tests

Run the test suite using pytest:

pytest tests/

Run with coverage:

pytest --cov=ncrads9 tests/

Code Style

NCRADS9 follows PEP 8 style guidelines. Use the following tools:

  • Black for code formatting
  • isort for import sorting
  • flake8 for linting
  • mypy for type checking

Run all checks:

black ncrads9/
isort ncrads9/
flake8 ncrads9/
mypy ncrads9/

Contributing

  1. Fork the repository
  2. Create a feature branch: git checkout -b feature/my-feature
  3. Make your changes and add tests
  4. Run the test suite to ensure all tests pass
  5. Commit your changes: git commit -m "Add my feature"
  6. Push to your fork: git push origin feature/my-feature
  7. Open a Pull Request

Building Documentation

Build the LaTeX documentation:

cd docs
make pdf

Architecture Overview

Core Components

  • FITSImage: Handles FITS file I/O and data management
  • Display: Manages image rendering and visualization
  • RegionManager: Handles region creation, editing, and serialization
  • WCSHandler: Provides coordinate transformations
  • AnalysisTools: Statistical and analytical functions

UI Framework

NCRADS9 uses PyQt5/PyQt6 for the graphical interface. The UI is organized into:

  • MainWindow: Primary application window
  • ImageCanvas: Image display widget
  • Dialogs: Various dialog windows for settings and tools
