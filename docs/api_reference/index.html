NCRADS9 API Reference

This document provides the complete API reference for NCRADS9.

Application Entry Points

ncrads9.__main__.main()

CLI entry point used by the ncrads9 console script.

ncrads9.app.run_application(argv: List[str]) -> int

Creates the Qt application, main window, optional XPA server, opens command-line files, and starts the event loop.

Core Data Modules

ncrads9.core.fits_handler.FITSHandler

Main FITS I/O helper used by the UI.

load(filepath)
get_data(ext=0)
get_header(ext=0)
list_extensions()

ncrads9.core.wcs_handler.WCSHandler

WCS coordinate conversion wrapper around Astropy WCS.

is_valid
pixel_to_world(x, y)
world_to_pixel(ra, dec)
get_pixel_scale()

UI and Frame Modules

ncrads9.ui.main_window.MainWindow

Main Qt window coordinating menus, display state, and analysis dialogs.

open_file(filepath=None)
_set_scale(...), _set_colormap(...), _set_bin(...)
_show_statistics(), _show_histogram(), _show_contours()

ncrads9.frames.simple_frame_manager.FrameManager

Manages frame creation, deletion, selection, and navigation used by blink/tile workflows.

Communication Modules

ncrads9.communication.xpa.xpa_server.XPAServer

DS9-style XPA server implementation.

start() -> bool
stop()
register_handler(command, handler)

ncrads9.communication.samp.SAMPClient

SAMP client wrapper used by VO menu actions for hub connect/disconnect and catalog overlays.

Rendering and Analysis Helpers

ncrads9.rendering.scale_algorithms

apply_scale(...)
compute_zscale_limits(...)

ncrads9.analysis.contour.ContourGenerator

Generates contour levels and paths used by the contour dialog/UI overlay.
