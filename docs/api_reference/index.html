NCRADS9 API Reference

This document provides the complete API reference for NCRADS9.

Core Module

class NCRADS9

The main application class for NCRADS9.

__init__(self)

Initialize the NCRADS9 application.

load_fits(filename: str, extension: int = 0) -> FITSImage

Load a FITS file into the viewer.

run(self) -> None

Start the application main loop.

class FITSImage

Represents a FITS image with associated metadata and WCS.

__init__(self, filename: str, extension: int = 0)

Load a FITS image from file.

data -> numpy.ndarray

Property: The image data array.

header -> astropy.io.fits.Header

Property: The FITS header.

wcs -> astropy.wcs.WCS

Property: World Coordinate System object.

Regions Module

class Region

Base class for all region types.

color: str

The region color (default: 'green').

width: int

Line width in pixels (default: 1).

to_ds9(coord_system: str = 'image') -> str

Export region to DS9 format string.

class CircleRegion(Region)

Circular region.

__init__(self, x: float, y: float, radius: float)

Create a circle region with center (x, y) and given radius.

class EllipseRegion(Region)

Elliptical region.

__init__(self, x: float, y: float, a: float, b: float, angle: float = 0)

Create an ellipse with center (x, y), semi-axes a and b, and rotation angle.

class BoxRegion(Region)

Rectangular region.

__init__(self, x: float, y: float, width: float, height: float, angle: float = 0)

Create a box with center (x, y), dimensions, and rotation angle.

Analysis Module

compute_statistics()

compute_statistics(data: numpy.ndarray, region: Region = None) -> dict

Compute statistics for the given data or region.

Returns: Dictionary with keys: sum, mean, median, std, min, max, centroid_x, centroid_y

compute_histogram()

compute_histogram(data: numpy.ndarray, bins: int = 100, range: tuple = None) -> tuple

Compute histogram of pixel values.

Returns: Tuple of (histogram_values, bin_edges)

radial_profile()

radial_profile(data: numpy.ndarray, center: tuple) -> tuple

Compute radial profile around a center point.

Returns: Tuple of (radii, profile_values)
